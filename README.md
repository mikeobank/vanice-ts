# Vanice-ts

Typescript implementation for https://github.com/mikeobank/vanice

## Run

```
./vanice { vanity name }
deno run src/main.ts { vanity name }
```

## Compile

```
deno compile --output vanice src/main.ts
```

## Development

```
deno lint
deno test
```
